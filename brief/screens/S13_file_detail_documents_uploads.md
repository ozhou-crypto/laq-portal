# S13 — File detail — Documents & uploads

**Personas:** All external and in-house practitioners
**User stories:** FR-B1-319, FR-B1-320

## Purpose

Upload and view supporting material in multiple formats, linked to the file.

## Layout

Drag-and-drop zone at top, document grid/list toggle below.

## Elements to include

- Drop zone accepting multiple files (PDF, DOCX, JPEG, PNG)
- Per-file upload progress with error recovery
- Document cards: thumbnail, name, type badge, size, uploaded-by, uploaded-at
- Inline preview for PDF and images; download for others
- Category tagging (e.g., QP9, Affidavit, Court order)

## States

Empty, uploading, mixed valid/invalid types, upload error.
