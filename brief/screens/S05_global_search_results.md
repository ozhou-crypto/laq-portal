# S05 — Global search results

**Personas:** All external and in-house users
**User stories:** FR-B1-308, FR-B1-311

## Purpose

Unified search across files, clients, correspondence, decisions, knowledge articles.

## Layout

Full-width results page with left filter rail and centre result stream. Predictive suggestions from the top-bar search.

## Elements to include

- Top search bar with predictive text dropdown (grouped: files, clients, knowledge)
- Filter rail: type, status, date range, firm location, Q-number
- Result cards with type badge, title, snippet with matched terms highlighted, metadata row
- Saved-searches panel
- Empty-state and zero-results treatments

## States

Predictive suggestions, results, filtered results, zero results, saved searches.

## Designer notes

Predictive text is mandated by FR-B1-311 — show it on both the top bar and in the dedicated results page.
