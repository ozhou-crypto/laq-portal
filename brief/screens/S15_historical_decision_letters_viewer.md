# S15 — Historical decision letters viewer

**Personas:** All external and in-house practitioners
**User stories:** FR-B1-327

## Purpose

Read-only archive of decision letters and related documents for the assigned firm.

## Layout

List + reader split. PDF inline viewer with download.

## Elements to include

- Letter list grouped by year, searchable by client / Q-number / keyword
- Reader pane with embedded PDF, metadata strip (decision date, officer, outcome)
- 'Acknowledge receipt' action where required
- Download and print

## States

Empty (new firm), populated, restricted (not the assigned firm — show clear why).
