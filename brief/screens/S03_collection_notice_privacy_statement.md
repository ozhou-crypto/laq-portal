# S03 — Collection Notice & Privacy Statement

**Personas:** All portal users
**User stories:** User Story #160

## Purpose

First-run modal and permanently linked page presenting LAQ's data collection and privacy obligations.

## Layout

Modal on first login; standalone page thereafter. Persistent footer link.

## Elements to include

- Scrollable statement text (plain-English summary first, full notice second)
- Last-updated date and version stamp
- 'I acknowledge' primary action and 'Print / download PDF' secondary action
- Persistent link from footer on every page

## States

First-run modal (blocking), standalone page, version-changed re-acknowledgement modal.

## Designer notes

Acknowledgement event must be captured in the audit log (see S23).
