# S12 — File detail — Correspondence tab

**Personas:** All external and in-house practitioners
**User stories:** FR-B1-321

## Purpose

Two-way portal messaging tied to the file, replacing ad-hoc email.

## Layout

Two-pane: message list (left) and thread view (right). Compose in modal or inline reply.

## Elements to include

- Thread list with unread badges, channel icon, LAQ participant avatars
- Thread view with message bubbles, attachments inline, timestamped
- Compose with light rich-text, attachment picker, recipient picker (pre-scoped to file)
- 'Linked to file' breadcrumb — make the file linkage visually unbreakable

## States

Empty, active thread, attachment upload in progress, message failed to send.
