# S14 — File detail — Outcomes (In-House focus)

**Personas:** In-House Solicitor (primary)
**User stories:** FR-B1-323

## Purpose

Electronic capture of matter outcomes so the grant file reflects legal progress.

## Layout

Tab within the file detail. Structured form plus outcomes timeline.

## Elements to include

- Outcomes timeline (chronological, most recent at top)
- 'Record outcome' form: outcome type, date, court (if relevant), notes, supporting document
- Outcome types reflect LAQ taxonomy (dropdown, not free text)
- Edit / cancel for recent outcomes within an allowed window

## States

Empty, populated, edit-in-progress, audit-locked (outside edit window).
