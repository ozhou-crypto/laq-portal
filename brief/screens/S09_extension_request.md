# S09 — Extension request

**Personas:** Preferred Supplier, In-House Solicitor, CLC, Authorised Support Staff
**User stories:** FR-B1-313, FR-B1-330

## Purpose

Submit an extension against an existing grant of aid, with the option to bundle an invoice submission.

## Layout

Modal-to-page hybrid: launched from a file (S11) as an overlay, expandable to full page for complex extensions.

## Elements to include

- File context panel (client, Q-number, current scope, conditions)
- Extension reason, anticipated additional hours/cost, supporting documents
- Optional 'Also submit invoice for work performed' toggle that expands an invoice panel
- Declaration and submit

## States

Standalone, combined-with-invoice, review, submitted.
