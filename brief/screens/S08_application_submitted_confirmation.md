# S08 — Application submitted confirmation

**Personas:** Preferred Supplier, CLC, In-House Solicitor, Authorised Support Staff
**User stories:** FR-B1-410

## Purpose

Terminal success screen issuing the unique reference number the practitioner can quote.

## Layout

Centred confirmation block with supporting action buttons.

## Elements to include

- Success tick (restrained, not celebratory — this is a government service)
- Unique reference number in prominent, copyable format
- What happens next (3-step timeline: received, assessed, decision)
- 'Email me a copy', 'Print receipt', 'Return to home', 'Start another application'
- Reminder that notifications will follow via the user's chosen channel

## States

Default only — no error variant (errors are handled in S06).
