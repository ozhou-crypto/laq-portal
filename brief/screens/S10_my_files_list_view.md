# S10 — My files — list view

**Personas:** All external and in-house practitioners
**User stories:** FR-B1-308

## Purpose

The practitioner's working inbox of grant files with strong search and filter.

## Layout

Full-width list view with saved list filters (Active, Archived, Awaiting action, Closed).

## Elements to include

- List tabs for status buckets
- Columns: Client, Q-number, Matter type, Status, Next action, Last activity
- Quick filter chips and advanced filter drawer
- Row hover reveals quick actions (Open, Extension, Invoice, Message)
- Density toggle (comfortable / compact), column chooser
- Export to CSV

## States

Active, archived, filtered, empty.
