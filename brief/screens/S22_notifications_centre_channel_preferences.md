# S22 — Notifications centre & channel preferences

**Personas:** All portal users
**User stories:** FR-B1-326

## Purpose

Central inbox of multi-channel notifications plus per-event channel preferences (SMS, email, in-portal).

## Layout

Two-pane: notifications list (left), detail with actions (right). Separate preferences page reached via header link.

## Elements to include

- Notification list grouped by today / earlier with type badges
- Detail pane with full context and deep links into file, invoice, etc.
- Bulk mark-as-read, filter by type
- Preferences page: matrix of event types × channels (in-portal, email, SMS) with per-row channel toggles
- Quiet-hours setting

## States

Populated, empty, preferences default, preferences edited.
