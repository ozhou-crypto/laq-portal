# Personas & Jobs To Be Done

Seven personas interact with the LAQ Practitioner Portal. When generating screens, use this document to inform tone, terminology, and what matters most to each persona.

## 1. Preferred Supplier (Panel Practitioner)
**Type:** External provider — private law firm
**Primary jobs:** Lodge applications reliably, get paid on time, manage many files simultaneously, minimise admin burden.
**Portal priorities:** Autosave, unique reference number on submission, electronic invoicing, status notifications, fast file search.

## 2. In-House Solicitor
**Type:** LAQ-internal employee
**Primary jobs:** Access files with existing LAQ identity (SSO), decline conflicted offers with structured reasons, record matter outcomes electronically.
**Portal priorities:** SSO (not MFA), structured decline-offer form, outcome capture, full visibility of historical decisions.

## 3. Community Legal Centre (CLC) Practitioner
**Type:** External provider — community legal centre
**Primary jobs:** Lodge applications efficiently with limited admin capacity, exchange correspondence securely, stay informed of file status.
**Portal priorities:** Pre-filled forms for existing clients, portal messaging linked to files, multi-channel notifications, Knowledge Base access.

## 4. External Barrister
**Type:** External provider — barrister on brief
**Primary jobs:** Receive and act on briefs, invoice for approved work, keep practice details current, get paid on time.
**Portal priorities:** Self-service ABN/banking updates, electronic invoice submission and cancellation, document uploads in common formats.

## 5. Family Dispute Resolution Practitioner (FDRP)
**Type:** External provider — accredited FDRP
**Primary jobs:** Receive briefs, upload outcomes, invoice for sessions, stay current with LAQ policy.
**Portal priorities:** Multi-channel notifications, portal-linked correspondence, invoice submission, CPD and article access.

## 6. Authorised Support Staff
**Type:** Delegated user (paralegal, legal secretary, admin)
**Primary jobs:** Prepare applications and invoices on the practitioner's behalf, work within firm licence limits.
**Portal priorities:** Delegated access approved by the practitioner, autosaving forms, visibility of file status.

## 7. LAQ System Administrator
**Type:** LAQ-internal — portal operations
**Primary jobs:** Ensure every action is traceable, manage licence allocation, evidence WCAG and privacy compliance.
**Portal priorities:** Comprehensive audit logging, licence cap enforcement with approval workflow, WCAG 2.2 AA conformance, collection notice acknowledgement.

---

## How to use this in screen generation

- If a screen is used by external providers, prioritise the clarity of status, next action, and reference numbers.
- If a screen is used by In-House solicitors only, use LAQ-internal terminology and assume SSO context.
- If a screen is used by Authorised Support Staff, make delegation clear (show "acting on behalf of" in the chrome).
- Admin screens (S23) should have visually distinct chrome so users cannot confuse them with practitioner screens.
