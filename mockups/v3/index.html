LAQ Practitioner Portal — v3 prototype

22 linked mockup screens covering every workflow in the screen inventory. Everything is wired: use the top navigation on any screen to jump between sections, or click through the end-to-end walkthrough below. Status states, form conditional logic, and the annotation toggle are all interactive — no code required.

End-to-end walkthrough — Preferred Supplier journey

S01 Sign in → S04 Dashboard → S06 Lodge application → S08 Confirmed → S11 File detail → S12 Correspond → S13 Documents → S14 Outcomes → S17 Invoice → S18 Invoice list
Brand notes. v3 matches legalaid.qld.gov.au: white site header with the logo tile in navy, a separate navy nav band underneath, white page backgrounds, navy-underlined tabs, and a navy footer. S01 and S02 use the Salesforce Identity / Experience Cloud login chrome (teal accent); S23 uses a distinct darker admin chrome. Build-approach badges show whether a region is OOTB Experience Cloud, an OmniStudio FlexCard, an OmniScript, a Salesforce Flow, a DataRaptor, or (rarely) a Custom LWC. Toggle them on/off on any screen using the ⚡ button in the header.
